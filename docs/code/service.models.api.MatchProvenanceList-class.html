Class MatchProvenanceList

source code

Class to represent a list of matches, as a response to an API request for match procenances matching a specified set of criteria.

It reflects back the original parameters of the request, and includes a list of serialised (to dict objects) MatchProvenance objects

{
    "since" : "<date from which results start in the form YYYY-MM-DDThh:mm:ssZ>",
    "page" : "<page number of results>,
    "pageSize" : "<number of results per page>,
    "timestamp" : "<timestamp of this request in the form YYYY-MM-DDThh:mm:ssZ>",
    "total" : "<total number of results at this time>",
    "matches" : [
        "<ordered list of MatchProvenance JSON objects>"
    ]
}
