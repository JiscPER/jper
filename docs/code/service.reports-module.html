admin_routed_report(from_date, to_date, reportfile)

source code 

Generate a (monthly) routed report from from_date to to_date. It is assumed that from_date is the start of a month, and to_date is the end of a month.

Dates must be strings of the form YYYY-MM-DDThh:mm:ssZ

Parameters:
  • from_date - start of month date from which to generate the report
  • to_date - end of month date up to which to generate the report (if this is not specified, it will default to datetime.utcnow())
  • reportfile - file path for existing/new report to be output
