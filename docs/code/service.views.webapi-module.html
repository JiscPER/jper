Process a list request, either against the full dataset or the specific repo_id supplied

This function will pull the arguments it requires out of the Flask request object. See the API documentation for the parameters of these kinds of requests.

Parameters:
  • repo_id - the repo id to limit the request to
Returns:
Flask response containing the list of notifications that are appropriate to the parameters
