Class FailedNotification

source code

Class which represents a notification that has been received into the system but has not been able to be routed to any repository accounts

It extends the BaseNotification and mixes that with the RoutingInformation, so see both of those class definitions for the data that is held.

This class is basically the same as the RoutedNotification, but exists to differentiate itself within the system so that it can be persisted separately.

This class also extends a DAO, which means it can be persisted.
